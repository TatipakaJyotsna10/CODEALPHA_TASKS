# Stock Portfolio Tracker

## Description

The Stock Portfolio Tracker is a simple Python project that helps users calculate the total value of their stock investments. Users can enter stock symbols and quantities, while stock prices are stored in a predefined dictionary. The program calculates the total investment amount and displays the portfolio value. It can also save the results to a text (.txt) or CSV (.csv) file for future reference.

## Features

* Accepts stock names and quantities from the user.
* Uses a hardcoded dictionary to store stock prices.
* Calculates individual stock values and total portfolio value.
* Displays investment summary.
* Optionally saves results to a `.txt` or `.csv` file.
* Beginner-friendly Python project.

## Technologies Used

* Python
* Dictionaries
* User Input/Output
* Basic Arithmetic Operations
* File Handling

## Example Stock Prices

stock_prices = {

## How It Works

1. Enter the stock symbol.
2. Enter the quantity of shares.
3. The program retrieves the stock price from the dictionary.
4. Calculates the investment value for each stock.
5. Displays the total portfolio value.
6. Optionally saves the report to a file.
