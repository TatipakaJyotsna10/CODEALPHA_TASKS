# CodeAlpha Basic Chatbot

## Description
This project is a simple rule-based chatbot developed using Python. The chatbot interacts with users through predefined responses based on user input. It demonstrates the use of basic Python concepts such as functions, loops, conditional statements, and input/output operations.

## Features
- Responds to common greetings like "hello"
- Answers simple questions such as "how are you"
- Exits the conversation when the user enters "bye"
- User-friendly command-line interface

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python chatbot.py
